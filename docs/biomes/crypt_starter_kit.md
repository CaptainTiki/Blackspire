# Crypt Biome - Starter Modular Kit

This document lists the first set of modular pieces we should aim to have for the crypt biome. Focused on what we need to build a small, playable multi-room section.

## Priority 1: Core Structural (Build the bones of the level)

| Piece                        | Suggested Sizes (in 16-unit increments) | Notes |
|-----------------------------|-----------------------------------------|-------|
| Straight Wall               | 64, 96, 128                             | Most common piece |
| Corner (Inner)              | -                                       | 90° turns |
| Corner (Outer)              | -                                       | 90° turns |
| Door Frame                  | 48 high × 32 wide                       | Current test size. May iterate. |
| Floor Tile / Panel          | 64×64, 32×32                            | Keep simple at first |
| Ceiling Tile / Panel        | 64×64                                   | Match floor for now |
| Simple Pillar / Column      | 32×32 footprint                         | Good for breaking up space and sightlines |

## Priority 2: Variation & Flow (Make rooms feel different)

| Piece                        | Suggested Sizes                  | Notes |
|-----------------------------|----------------------------------|-------|
| Archway                     | 64 high                           | Strong visual element |
| Wall Niche / Alcove         | 48–64 high                        | Perfect for urns, torches, small props |
| Raised Platform             | 16 high                           | Good for verticality without big height changes |
| Rubble / Damaged Wall       | Various                           | Helps sell age and wear |
| Wooden Support Beam         | Various lengths                   | Breaks up pure stone look |

## Priority 3: Gameplay Props (Make the space feel interactive)

| Piece                        | Notes |
|-----------------------------|-------|
| Wall Torch                  | Both static and with light source |
| Floor Brazier / Torch       | - |
| Breakable Urn / Pot         | Multiple variants |
| Coffin                      | Some openable, some breakable |
| Lever / Switch              | Wall or floor mounted |
| Bone Pile / Skull Cluster   | Good filler and enemy emergence points |
| Iron Grate / Gate           | Can block or be opened |

## Recommended First Pass (MVP Kit)

If we want to block out a small 3–5 room crypt section quickly, aim for these pieces first:

1. Straight Wall (64 + 96)
2. Inner + Outer Corners
3. Door Frame (48h × 32w)
4. Floor + Ceiling panels (64×64)
5. Simple Pillar (32×32)
6. Wall Niche
7. Wall Torch (with light)
8. Breakable Urn (at least 1–2 variants)
9. Coffin (basic)

This small kit should let us build a decent-feeling loop with some vertical contrast (64 vs 48 doorways) and a few interactables.

## Notes

- All pieces should be built with the **64-unit module** and **16-unit grid** in mind.
- Focus on clean, readable silhouettes first. Detail can come later.
- Think about how pieces will be lit (torch placement, shadow-casting edges, etc.).
- We can add more ornate carved pieces once the basic kit feels good.
